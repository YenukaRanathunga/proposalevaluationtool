# GBV Mitigation - Proposal Evaluation Tool

A professional web-based tool for evaluating consultancy proposals using weighted criteria matrix. Designed for procurement committees to streamline the evaluation process for GBV (Gender-Based Violence) mitigation projects.

### Core Functionality
